Por que estudar JavaScript?

JavaScript é uma linguagem bastante poderosa que permite que o usuário consiga interagir com a página.Com certeza isso já seria um excelente motivo para despertar seu interesse pela linguagem, mas além de ser dominante nos cenários dos navegadores.O JS recentemente também se expandiu para o lado do servidor com o lançamento do Node.js, um ambiente que permite criarmos um servidor Web completo utilizando JavaScript.Desta forma, a linguagem ganhou grande popularidade e caiu no gosto dos desenvolvedores, tornando-se parte do coletivo imaginário de qualquer programador moderno.

JavaScript é uma das 3 linguagens que todos os desenvolvedores da web devem aprender:

  • HTML para definir o conteúdo das páginas da web
  • CSS para especificar o layout das páginas da web
  • JavaScript para programar o comportamento das páginas da web
